# 강아지 벅벅 3D 🐶 (품종별 강아지 교감 게임)

**9가지 실제 품종** 중 우리 강아지를 골라, 다양한 도구로 만지며 교감하는 브라우저 힐링 게임입니다.
설치·서버·인터넷 전부 필요 없습니다. `index.html` 더블클릭이면 끝.

---

## ▶️ 실행 방법

1. `index.html`을 **더블클릭** → 크롬/엣지에서 바로 열립니다.
2. 강아지를 마우스(또는 손가락)로 누른 채 문질러 보세요.
3. **소리는 첫 클릭/터치 이후에 켜집니다.** (브라우저 자동재생 정책)

## 🖱️ 조작법

| 입력 | 동작 |
|---|---|
| **좌클릭 드래그** | 선택한 도구로 강아지 만지기 (벅벅!) |
| **우클릭 (바닥)** | 클릭한 곳으로 강아지가 **타닥타닥 달려감** — 몸을 돌리고 트로트 걸음, 도착하면 카메라 쪽으로 돌아봐요. 달리는 중에 다시 우클릭하면 경로 변경 |

## 🐶 품종 선택 (커스터마이징)

우측 상단 품종 버튼 → 9품종 카드에서 클릭하면 **즉시 변신**합니다. 선택은 자동 저장돼요.

| 품종 | 외형 포인트 | 성격 (게임에 반영) |
|---|---|---|
| ☁️ 말티즈 | 순백 털, 접힌 귀, 등 위로 말린 꼬리 | 애교만렙 — 행복·편안함이 빨리 차요 |
| 🦊 포메라니안 | 풍성한 목털 솜뭉치, 여우 주둥이, 큰 플룸 꼬리 | 씩씩하고 잘 짖어요 |
| 🐕 시바견 | 쫑긋 삼각귀, 말린 꼬리, 우라지로(흰 배·볼) | 독립적·도도 — 발라당을 잘 안 해줘요 |
| 🍞 웰시코기 | 긴 몸 + 숏다리, 큰 귀, 짤막 꼬리 | 꼬리 대신 **엉덩이 전체가 씰룩** |
| 🌭 닥스훈트 | 소시지 몸매, 긴 주둥이, 늘어진 귀, 블랙&탄 눈썹점 | 용감하고 호기심 많아요 |
| 🐾 비글 | 삼색(갈·검·흰) + 등 안장무늬, 흰 꼬리끝, 큰 귀 | 에너지 폭발, 간지럼 많이 타요 |
| 🎀 푸들 | 롱다리, 곱슬 토크노트, 방울 꼬리 | 우아한 천재견 |
| 🐺 시베리안 허스키 | **파란 눈**, 흰 얼굴 마스크, 낫모양 꼬리, 대형 | 시크 — 간지럼에 둔감, 흥분은 빨라요 |
| 🌟 골든 리트리버 | 황금빛 대형견, 접힌 귀, 깃털 꼬리 | 천사견 — **발라당을 제일 잘해요** |

품종 성격(`mind`)이 감정 시스템 배율로 들어가서, 같은 도구·부위라도 품종마다 반응 속도와 정도가 다릅니다.

## 🎮 부위 × 도구 교감 시스템

**부위 5곳** — 머리 · 몸 · 다리 · 꼬리 · 엉덩이:

| 부위 | 반응 |
|---|---|
| 머리 | 귀 쫑긋 + 눈 지그시 + 고개 갸웃 |
| 몸 | 젤리처럼 출렁 + 계속 만지면 **발라당 뒹굴기** |
| 다리 | 파닥파닥 + 콩콩 뛰기 (간지럼!) |
| 꼬리 | 살랑살랑 → 프로펠러 붕붕 |
| 엉덩이 | 씰룩씰룩 + **뒷발 탁탁 긁기반사** |

**도구 5종** — 🖐️ 손 / 🪮 빗(털 윤기) / 🪶 깃털(간지럼) / 🥄 숟가락(시원한 벅벅) / 📳 마사지건(대고만 있어도 진동, 노곤노곤)

**감정 시스템**: 행복·편안·흥분·졸림이 실시간으로 변하고, 가만 두면 잠들어요(Zzz). 신나면 진짜 멍멍 소리.
**유대감 레벨**: 품종마다 따로 쌓이고, 레벨업하면 하트 폭죽 🎉

## 📁 파일 구조

```
강아지 벅벅_2/
├─ index.html        ← 게임 진입점 (이거 더블클릭)
├─ css/style.css     ← 포근한 UI + 품종 카드 스타일
├─ js/
│   ├─ three.min.js  ← Three.js (3D 엔진, r128)
│   ├─ audio.js      ← 사운드 합성 (벅벅 ASMR·배경음·효과음·멍멍)
│   ├─ breeds.js     ← 품종 9종 정의 (체형·귀·꼬리·무늬·성격) ★커스터마이징
│   ├─ tools.js      ← 도구 5종 + 부위별 반응 매트릭스
│   ├─ dog3d.js      ← 로프트(단면 스킨) 3D 강아지 + 버텍스 컬러 무늬 ★핵심
│   ├─ input.js      ← 긁기 입력 (마우스+터치)
│   └─ game.js       ← 씬·배경·품종 선택 UI·메인 루프
└─ models/sounds/dog.ogg  ← 멍멍 소리 (Wikimedia Commons, CC BY-SA 3.0)
```

## 🔧 느낌 바꾸기 (튜닝 포인트)

| 바꾸고 싶은 것 | 위치 |
|---|---|
| 품종 추가/체형·색·성격 수정 | `js/breeds.js` 의 `BREEDS` (항목 복사해서 새 품종 추가 가능) |
| 긁기 민감도 | `js/input.js` 의 `SPEED_MAX` |
| 부위별 감정 반응 | `js/dog3d.js` 의 `ZONE_FX` |
| 도구별 세기/대사/음색 | `js/tools.js` 의 `TOOLS` |
| 유대감 레벨업 난이도 | `js/game.js` 의 `needFor` |
| 방 색감/창문 | `js/game.js` 의 `drawBackground` |
